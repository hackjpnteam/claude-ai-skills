# Claude AI Skills Collection

hackjpn株式会社 AI研修プログラム用スキルコレクション

## 概要

Claude Code で使える実践的なスキル（カスタムコマンド）のコレクションです。
システム開発、広告運用、営業資料作成などの業務を AI で自動化します。

## スキル一覧

### コマンド（`commands/`）— `/コマンド名` で起動

| コマンド | 説明 |
|---------|------|
| `/dev` | フルスタック開発 → デプロイ → 納品書を一気通貫で実行 |
| `/contract` | 電子契約書の作成・PDF生成・メール送信 |
| `/ad` | Meta/X/Shopify 広告の分析・入稿・最適化 |
| `/finance` | 財務データの確認・分析・レポート生成 |
| `/msg` | クライアント向けビジネスメッセージ生成 |
| `/remember` | Git履歴から過去の開発内容を復元 |
| `/long` | 長文プロンプトの安全な入力支援 |
| `/test` | QA テスト手順の自動生成 |

### スキル（`skills/`）— 自動トリガーまたは `/スキル名` で起動

| スキル | 説明 |
|--------|------|
| `slide` | Reveal.js 営業スライド自動生成 → Vercel デプロイ |
| `pdf` | URL/HTML → 高画質PDF変換 |
| `qa` | コードベース品質監査（5並列エージェント） |
| `ad` | Meta広告 高速分析 |
