# 📊 Spreadsheet - データ分析スプレッドシート自動作成スキル

## 概要
プロジェクトのユーザーデータを自動分析し、Google Sheetsに反映させて共有可能なスプレッドシートURLを生成するスキルです。

## 主要機能

### 1. **自動データ分析**
- CSVファイルからユーザーデータを読み込み
- 顧客カテゴリの自動分類（有料・招待無料・一般無料・テスト）
- 収益指標の自動計算
- 地域・職業別の統計分析

### 2. **Google Sheets自動生成**
- 統合分析レポートの自動作成
- ビジネス戦略・成長提案の生成
- 共有可能なURLの即座発行
- カテゴリ別詳細リストの作成

### 3. **ビジネス洞察の自動生成**
- 転換率分析と業界比較
- 収益最適化提案
- 顧客獲得戦略の提案
- 具体的アクションプランの作成

## 使用方法

### 基本実行
```
/spreadsheet
```
現在のプロジェクトのユーザーデータを分析し、Google Sheetsに反映

### CSVファイル指定
```
/spreadsheet data.csv
```
特定のCSVファイルを指定して分析

### カスタム分析
```
/spreadsheet --detailed
```
より詳細な分析とインサイト生成

## 自動実行フロー

### Phase 1: データ発見・読み込み
1. **CSV自動検出**
   ```bash
   find . -name "*user*" -name "*.csv" | head -5
   ```
2. **データ構造解析**
   - ヘッダー情報の確認
   - データ品質チェック
   - 不正データの除外

### Phase 2: 顧客分析・カテゴリ分類
1. **顧客セグメンテーション**
   - 有料顧客: 課金状況="課金中"
   - 招待無料: Tier=99 または プロモーションコード有
   - 一般無料: 通常の無料登録ユーザー
   - テストアカウント: 開発用アカウントの自動識別

2. **収益分析**
   - MRR（月間経常収益）計算
   - ARR（年間想定収益）算出
   - LTV（顧客生涯価値）計算
   - 転換率の業界比較

### Phase 3: ビジネス洞察生成
1. **成長戦略分析**
   - 地域別市場ポテンシャル
   - 職業別ターゲティング提案
   - 競合比較と差別化戦略

2. **収益最適化提案**
   - Tier別アップセル戦略
   - チャーン防止施策
   - 価格戦略の最適化

### Phase 4: Google Sheets作成
1. **統合分析シート**
   ```javascript
   // 自動的に以下を含む統合レポート作成
   - 概要統計
   - 有料顧客リスト（名前・メール・地域・職業・事業名込み）
   - 招待無料ユーザーリスト
   - 一般無料ユーザーリスト（転換可能性評価込み）
   ```

2. **ビジネス戦略シート**
   ```javascript
   // 自動的に以下を含む戦略レポート作成
   - 短期アクション（1-3ヶ月）
   - 中期戦略（3-6ヶ月）
   - 長期展望（6-12ヶ月）
   - 具体的KPI目標
   ```

3. **共有URL生成**
   - Google Drive APIによる自動共有設定
   - 編集権限の適切な設定
   - アクセス可能なURL即座発行
